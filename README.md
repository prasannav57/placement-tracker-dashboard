# PSNA Placement Tracker

A placement tracker dashboard for PSNA students and placement cell staff. This project helps students check placement eligibility and allows admins to manage company criteria through a simple dashboard.

## Features

- Student eligibility checker
- Required student details:
  - Name
  - Register number
  - Resume link
  - CGPA
  - Branch
  - Arrears
- Company management dashboard
- Add, edit, and delete company rules
- Node.js and Express backend
- File-based company data storage

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express
